# dll_invoke
1. 安装依赖
pip install -r requirements.txt
2. 使用方法
2.1 查看函数列表
1. 打开cmd窗口
2. 输入命令：`curl -H "apiKey: a1234567890" http://127.0.0.1:5000/functions`
3. 输出：
[
  {
    "func_id": 1,
    "func_name": "GD_HYDROSTATIC_PRESSURE"
  }
]
